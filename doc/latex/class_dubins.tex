\doxysubsubsection{Detailed Description}
\doxylink{class_dubins}{Dubins} path used to calculate the path between two points in the city graph. 

This class represents a \doxylink{class_dubins}{Dubins} path used to calculate the path between two points in the city graph. Given the start and end points, it calculates the path, the distance and the time to reach the end point.
